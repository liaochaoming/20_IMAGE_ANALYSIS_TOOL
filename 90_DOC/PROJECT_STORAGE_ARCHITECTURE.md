## 目的

本文件說明圖像分析工具如何依 `00_KINGDOM_SYS.md` 建立本機運算、NAS 保存的專案結構。

## 結論

```text
HOST 產生，NAS 保存，HOST 鏡像備援
```

因此：

- `D:\20_IMAGE_ANALYSIS_TOOL` 是 HOST_B 本機工作區。
- `30_CATALOG` 內的 `00_DATA / 20_ANALYSIS_RESULT / 30_DB / 40_RAG / 50_TAG` 將來會搬到 NAS 作為正式資料。
- 本機可保留快取與執行暫存。
