## 目錄用途

- 10_SYS：開機必讀規則與專案狀態。
- 20_CONFIG：全系統共用 schema、分類、標籤、儲存策略。
- 30_CATALOG：四大分類目錄，包含來源標準、分類規則、分析入口、輸出、DB、RAG、TAG、文件。
- 40_ALL_INPUT：全域未分類入口，使用者雜亂上傳圖片先進此處。
- 70_SCRIPT：安裝、分析、維護腳本。
- 80_APP：WebUI / API / Gradio / FastAPI 應用，目前包含 ANALYSIS_ENGINE 與 ASSET_MANAGER_WEBUI。
- 90_DOC：中文說明、架構文件、索引文件。
- 99_LOG：執行紀錄與排錯 log。
